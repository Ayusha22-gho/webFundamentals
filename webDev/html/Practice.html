We would learn about HTML-Hyper Link Markup Language

We would learn about the following things:-

  1. Declaration
  2. TAGS
  3. ATTRIBUTES
  4. Quotations and Citations
  • Declaration is DOCTYPE which shows which language are we using, here we are using !DOCTYPE html along with enclosing tags '<' and '>' representing the html language we would use.
  • TAGS are the different keywords we use in order to perform various functions in our website ,like 'h1'is a heading tag which also needs to be closed using '/h1'
  • Attributes are part of the tags which adds extra feature to our text ,like style,bgcolor ,backgroung,title etc. The syntax is: tag attribute="feature:value"
